🍜 EATLOOP - Food Delivery Platform

 Tech Stack
- React 18, React Router, Recharts
- Node.js, Express.js, MongoDB
- JWT Authentication, Socket.io
- AI/ML: Recommendation Engine, Sentiment Analysis

 Features
- Real-time order tracking with Socket.io
- AI-powered food recommendations
- Multi-role system (Customer/Restaurant/Delivery/Admin)
- Payment integration (Stripe + Razorpay)
- Sentiment analysis on reviews
- Demand forecasting for restaurants
